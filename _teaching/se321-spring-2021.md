**Description**

This introductory course will help to understand the application of software engineering principles and techniques to the development, deployment, and maintenance of high-quality Web-based systems and applications. The course addresses the concepts, methods, technologies, and techniques of developing Web sites that collect, organize and expose information resources. Topics covered include requirements engineering for Web applications, design methods and technologies, interface design, usability of web applications, accessibility and security of Web Applications. Specific technologies covered in this course include client-side (HTML, JavaScript, and CSS) and server-side (PHP &amp; MySQL).

**Course Objectives**
* To be able to analyze and design comprehensive systems for the creation, dissemination, storage, retrieval, and use of electronic records and documents.
* To learn and use client-side and server-side languages used to manipulate information on the World Wide Web.
* To learn techniques for ensuring the proper maintenance and security of a web application.

**Course Learning Outcomes (CLO)**
* CLO1: Understand the concepts, principles and methods of Web Applications.
* CLO2: Gain the ability to design modern websites using the Frontend Technologies.
* CLO3: Obtain the ability to develop web applications using client-side &amp; server-side programming languages and components.
* CLO4: Apply the web engineering methodologies for Web application development.
* CLO5: Present and defend technical aspects of modern web engineering.
